PROJECT

Hotel Reviews

CATEGORY
WEBSITE PORTAL
LANGUAGE
HTML, CSS, PHP, MySQL
SOFTWARE USED
NetBean

A website called Hotel Reviews allows users to log in and enter reviews for individual hotels. They'll support an increase in foot traffic and reservations, as well as provide visitors more assurance in their choice to stay at the facility.

This website was created using HTML for the website's structure, CSS for decoration, PHP to check for validation, and MySQL to connect the websites to databases. The user can browse the hotels and read all of the reviews, but in order to post their own reviews, they must first register and log in.
